# Padeline

Backend de una aplicación de gesitón de clubs y pistas de pádel, con usuarios y partidos.


## Test cases
* Admin
1) Login
2) Obtener todos los usuarios
3) Crear un Manager

* Manager
1) Login
2) Intentar crear otro manager (no permitido)
3) Crear Club
4) Crear Pista

* Player
1) Login
2) Edit profile
3) Create Match
